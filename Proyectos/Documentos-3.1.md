# Documentos 3.1

Sistema de generación legal de documentos multi-usuario.

## Cobertura

- **17 templates legales:** 8 de compra, 9 de alquiler
- **100+ cláusulas dinámicas**

## Stack

- GitHub Pages (frontend)
- Vercel Functions (backend serverless)
- Supabase (base de datos + auth)
- Google OAuth2 (autenticación)
